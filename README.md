# Image Locker

## Install

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Features

- Upload image by click and drag.
- Filters
  - Grayscale
  - Invert
  - Sepia
- Download image.
- Copy Base64 to clipboard.
- Lock image with password.
- Unlock image with password.
- Crypto with AES-256-GCM
